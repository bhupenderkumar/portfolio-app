# Bhupender Kumar - Portfolio

A stunning portfolio website built with React, TypeScript, Vite, Tailwind CSS, and Framer Motion.

## Features
- Interactive particle background
- Smooth animations
- Responsive design
- Dark mode
- Skills visualization
- Experience timeline
- Project showcase

## Tech Stack
- React 18
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion

## Run Locally
```bash
npm install
npm run dev
```
